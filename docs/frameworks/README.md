# Framework guides

Esta librería soporta dos modos:

1. **React API** (`DominicanRepublicMap`)
2. **Web Component** (`<dr-map>`) para Vue, Svelte, Angular y Vanilla JS

## Guías

- [React](./react.md)
- [Vue](./vue.md)
- [Svelte](./svelte.md)
- [Angular](./angular.md)
- [Vanilla JS](./vanilla.md)

## Eventos del Web Component

`<dr-map>` emite:

- `provinceclick`
- `provincedoubleclick`
- `provinceenter`
- `provinceleave`
- `selectionchange`
- `markerclick`
- `zoomchange`
- `mapclick`
